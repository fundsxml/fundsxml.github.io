        Control Data

        Control Data are Metadata describing the data transmitted within the file.
        This Data contain information about the file itself (creation time, FundsXML Version, etc.) and the sender of the file (Data Supplier).
        The sender of the file does not have to be the manager of the fund. This can also be a technical service provider.
