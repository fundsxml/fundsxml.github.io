The ISOCountryCodeType is a type for decoding ISO-CountryCodes. It is used
the two-letter ISO-CountryCodes (ISO 3166-1alpha-2). It is of the type string and has a length from
exactly two letters.

Der ISOCountryCodeType stellt einen einfachen Typ zur Verarbeitung von
