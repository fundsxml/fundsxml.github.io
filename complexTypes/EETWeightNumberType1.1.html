Complex Type: EETWeightNumberType1.1

Child Elements

Name Type Cardinality Documentation
Weight PercentageType 0..1

The PercentageType defines the percentage-values. It is derived from
decimal. To decimal there is no modification. Percentage values can be expressed in three was: For 3.5 %
it can be written as 3.5 or 1.035 or 0.035. For FundsXML percentage value has to be written as 3.5

Der PercentageType definiert die Prozentwerte. Abgeleitet ist er von
decimal. Veraendert wurde er nicht. Bei Prozentwerten kann man je nach Definition fuer 3, 5 % die Werte
3.5 oder 1.035 oder 0.035 angeben. Mit diesem Typ dokumentiert FundsXML, dass fuer Prozentwerte die
Auspraegung 3.5 benutzt werden muss um eine einheitliche Auspraegung fuer den Datenaustausch zwischen
Firmen zu erreichen.
