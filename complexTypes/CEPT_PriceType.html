Share_price_factor xs:decimal 0..1

11040-Factor as combination of distribution value and share split factor (if this
field is provided then following fields are not required). Value which the price should be
multiplied to to obtain the correct performance time series. This should reflect the multiplying
factor and should be populated every day. In case of null value, '1' is taken as default. Should
only be populated in case a share price factor is relevant.
DistributionValue xs:decimal 0..1

11050-Distribution value in share price currency. Should be populated on day of
distribution only and only in case a distribution has taken place.
ShareSplitFactor xs:decimal 0..1

11060-Should only be populated in case of a share split - then respective factor
should be populated. This should reflect the multiplying factor and should be populated every
day. In case of null value, '1' is taken as default.
Old_share_class_currency ISOCurrencyCodeType 0..1

11070-In the rare occasion of a share class currency change - this needs to be
populated with the old share class currency. Should only be populated in case a share class
currency change took place.
