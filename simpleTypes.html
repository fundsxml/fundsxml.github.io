PercentageType
The PercentageType defines the percentage-values. It is derived from decimal. To decimal there is no modification. Percentage values can be expressed in three ways: For 3.5 % it can be written as 3.5 or 1.035 or 0.035. For FundsXML percentage value has to be written as 3.5 Der PercentageType definiert die Prozentwerte. Abgeleitet ist er von decimal. Bei Prozentwerten kann man je nach Definition für 3,5 % die Werte 3.5 oder 1.035 oder 0.035 angeben. Mit diesem Typ dokumentiert FundsXML, dass für Prozentwerte die Ausprägung 3.5 benutzt werden muss um eine einheitliche Ausprägung für den Datenaustausch zwischen Firmen zu erreichen. PercentageType définit les valeurs en pourcentage. Il est dérivé de decimal. Par rapport à decimal, il n'y a aucune modification. Les valeurs en pourcentage peuvent être exprimées de trois façons: Pour 3,5%, on peut écrire 3.5 ou 1.035 ou 0.035. Pour FundsXML, la valeur en pourcentage doit être écrite comme 3.5 PercentageType definieert percentagewaarden. Het is afgeleid van decimal. Ten opzichte van decimal is er geen wijziging.
